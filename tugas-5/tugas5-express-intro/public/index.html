Express.js Application Documentation

Endpoints

GET /hello

Returns a success message with "Hello World!".

Response:
{
  "message": "Success fetch message",
  "data": "Hello World!"
}

GET /user

Returns user details.

Response:
{
  "message": "Success fetch user",
  "data": {
    "id": 1,
    "name": "Budi",
    "username": "budidu",
    "email": "budidu@mail.com"
  }
}
